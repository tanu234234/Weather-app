# API Integration – Weather App
📖 Project Description

This project is a dynamic and responsive Weather Application developed using HTML, CSS, and JavaScript that integrates a public weather API to fetch and display real-time weather information. The main objective of this project is to demonstrate API integration in a web application and show how real-time external data can be retrieved, processed, and displayed in an interactive user interface.

The application allows users to search for the weather conditions of any city in the world. Once the user enters the name of a city, the system sends a request to the Open-Meteo Geocoding API to obtain the geographical coordinates (latitude and longitude) of that city. These coordinates are then used to call the Open-Meteo Weather API, which returns the current weather data including temperature and wind speed.

The weather data received from the API is processed using JavaScript and dynamically displayed on the webpage without requiring a page refresh. This demonstrates the concept of asynchronous data fetching using APIs, which is an essential skill in modern web development.

One of the key highlights of this application is its visually appealing dynamic background system. Based on the weather conditions returned by the API, the application automatically changes the background animation. For example, when the weather is sunny, the interface displays a bright sunny background; when it is cloudy, animated clouds appear; and when it is rainy, rain animations are shown. This feature improves the overall user experience and makes the application more interactive and engaging.

The application also includes basic error handling and loading states. If a user enters an invalid city name or if the API fails to return data, the system provides appropriate feedback instead of crashing. This ensures a smooth and reliable user experience. Additionally, the interface is designed to be clean, simple, and responsive, meaning it can adapt to different screen sizes and devices.

This project demonstrates several important web development concepts including API integration, asynchronous JavaScript, DOM manipulation, responsive UI design, and dynamic content rendering. It also shows how front-end technologies can interact with external data sources to create useful real-world applications.

Overall, the Weather App serves as a practical example of how APIs can be integrated into web applications to provide real-time data and interactive functionality. It is a beginner-friendly project that helps developers understand how external services can be used to enhance web applications.

# How it Works
The user enters the name of a city in the search input field.
The application sends a request to the Geocoding API to get the city's coordinates.
Using these coordinates, another request is sent to the Weather API.
The API returns weather details such as temperature and wind speed.
JavaScript processes the data and dynamically updates the webpage.
The background animation changes depending on the weather condition.
