
# 🌦 Weather App

A dynamic and responsive Weather App that fetches real-time weather data using a public API. The app allows users to search for any city and displays current temperature and wind speed with visually appealing animated backgrounds.


## 🚀 Features

- 🌍 Search weather for any city
- 🌡 Displays real-time temperature
- 💨 Shows wind speed
- 🎨 Dynamic background based on weather conditions:
  - ☀️ Sunny
  - ☁️ Cloudy (with moving clouds)
  - 🌧 Rainy (with rain animation)
- ⚡ Loading and error handling
- 💻 Clean and responsive UI

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Open-Meteo API (Geocoding + Weather API)

Weather-app/

## 🔗 How it Works

1. User enters a city name  
2. App fetches coordinates using Geocoding API  
3. Weather data is fetched using Open-Meteo API  
4. UI updates dynamically with weather info and animations
